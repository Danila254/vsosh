# Программа для обучения по криптографии

Запуск программы в файле [main.py](https://github.com/Danila254/vsosh/blob/main/main.py).

Алгоритм работы программы:
1. Выбор шифра.
2. Набор ключа.
3. Ввод слова , которое необходимо зашифровать.
4. Зашифровка.
